Manager Rest Api 汇总

Manager Rest Api 目录

  • signIn – 登录接口
  • signUp – 注册接口
  • userDetail – 用户信息接口
  • goodsList – 商品列表接口
  • goodsDetail – 商品信息接口
  • pay – 支付接口
  • orderList – 订单列表接口
  • rushToBuy – 抢购接口
  • pendingPayment – 获取未付款订单列表接口
  • payPendingPayment – 对未付款订单进行付款接口

Api 接口返回码说明

    成功：ManagerService.200

    失败：ManagerService.400

Api 接口详细介绍

1）、登录接口

  • rest接口：v1/rest/signIn
  • 请求类别：post
  • 请求参数：
{

    "userName":"jack",

    "userPwd":"123"

}
  • 返回结果：
{

    "errCode":"ManagerService.200",

    "resMsg":[

        {

            "userId":5,

            "userName":"Jack"

        }

    ],

    "signInResult":"SignInSuccess"

}

2)、注册接口

  • rest接口：v1/rest/signUp
  • 请求类别：Post
  • 请求参数：

 }
